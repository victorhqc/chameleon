chameleon
=========

Get the main color of a picture (Inspired in the chameleonic feature in Ubuntu).

How to use
----------

```
new Chameleon({
	image: 'some_image.jpg',
	callback: function(e){
		console.log('e', e);
	}
});
```

Installation
------------
Just include the `chameleon.class.js` file
